# ManeKelsa
helps local worker with daily wages 
````md

ManeKelsa is an Android-based job and career support application developed using Android Studio.  
The application helps users explore job opportunities, manage career-related information, and access a simple and user-friendly platform for employment support.

---

## Features

- User Registration and Login
- Job Search Functionality
- Easy Navigation and User-Friendly UI
- Firebase Database Integration
- Real-Time Data Handling
- Secure Authentication System
- Responsive Android Application

---

## Technologies Used

- Android Studio
- Java / Kotlin
- XML
- Firebase Authentication
- Firebase Realtime Database / Firestore
- Git & GitHub

---

## Project Structure

```text
app/
gradle/
build.gradle
settings.gradle
AndroidManifest.xml
```

---

## Installation and Setup

1. Clone the repository

```bash
git clone https://github.com/AiraShield/ManeKelsa.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Connect Firebase (if required)

5. Run the application on Emulator or Android Device
````
